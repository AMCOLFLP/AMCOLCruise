This edition applies the same premium black, deep-navy, white, and metallic-gold design used by the AMCOL Maritime English for Engineers portal. The original cruise-ship course content and interactive functions are retained.

Open login.html to begin.

AMCOL Comprehensive Maritime English for Cruise Ships Learning Portal

START
1. Keep all HTML files and the complete assets folder together.
2. Open login.html in a modern browser.
3. For installable PWA behavior, serve the folder through a local web server.

- Administrator: admin / AMCOLadmin
- Teacher: teacher / AMCOLteacher

COMPREHENSIVE COURSE STRUCTURE
The portal contains seven units and forty-nine paginated unit pages. Every unit now includes:
- Expanded overview, essential question, prerequisite knowledge, and performance standard
- Detailed learning outcomes and prior-knowledge activity
- Twenty-four cruise terms plus collocations, word families, pronunciation guidance, and vocabulary production
- Five expanded operational concepts
- Three-paragraph cruise reading passage with ten comprehension questions
- Four grammar and communication patterns
- Professional phrase banks and weak-versus-professional language comparison
- Model workplace dialogue
- Audio player, speed controls, replay, transcript, and listening questions
- Model handover, announcement, report, alert, briefing, or service-recovery document
- Expanded case study, operational data, stakeholder map, and seven-step decision process
- Infographic and comic scenario
- Applied performance activity, educational decision tool, output template, and rubric
- Ten unit-specific multiple-choice questions, short answers, applied case, professional communication task, and self-assessment
- Teacher answer key and scoring guidance
- 70% objective-check requirement for unit completion

ROLE ACCESS
- Administrator: all course content plus local page editing and management prototypes

- Student: learning units, listening lab, glossary, activities, assessments, resources, and capstone; restricted teacher/admin content remains hidden

SECURITY
This is an offline classroom prototype using browser localStorage. Public deployment requires secure server-side authentication, password hashing, protected sessions, a database, and server-side authorization.


DETAILED DISCUSSION EXPANSION (2026)
Every one of the 49 unit-section pages now contains a new guided teaching layer. The expansion adds plain-language explanations, teacher talk, step-by-step operational processes, role boundaries, model messages, common-error correction, worked scenarios, grammar form-meaning-use support, listening and writing scaffolds, case-analysis models, task checklists, review maps, and remediation guidance. Existing page navigation, role restrictions, audio, autosave, assessments, and offline behavior are preserved.
